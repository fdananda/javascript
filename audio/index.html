Audio play() Method / Audio pause() Method

O método audio.play() inicia a reprodução do áudio atual.

O método audio.pause() interrompe (pausa) o áudio que está sendo reproduzido.

Exemplo: 

	let meuaudio = document.getElementById("meuAudio");

	function iniciarAudio() {     
      		meuaudio.currentTime = 0;
      		meuaudio.play();
	}

	function pararAudio() {     
      		meuaudio.pause();
	}
