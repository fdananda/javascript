isNaN()

O método isNaN() verifica se um valor não é um número ("Not-a-Number"). O isNaN() retorna true se um valor não for um número e false se for um número. O método primeiro converte o valor em um número antes de testá-lo.

Exemplo: 

function checarSeEUmNumero() {   

    let saida = document.getElementById("exemplos")

    let nome = "Fernanda"
    let nascimento = 1977
    let nan1 = isNaN(nome)
    let nan2 = isNaN(nascimento)

    saida.innerHTML = `A variável nome contém o valor ${nome} e não é um número: ${nan1}
    A variável nascimento contém o valor ${nascimento} e não é um número: ${nan2}`
}
